My projects

Flutter Mobile App Development

  • WorldTime

    • My first Flutter application. It is an application used to view the time in cities around the world. The background of the app changes with the time of the day: morning, afternoon, evening and night adding to the application's visual appeal. The application allows users to pick their default location, the city's time will be displayed whenever the app is opened. Further, the application allows users to favourite locations in order to be able to access those cities easily.

  • BrewCrew

    • I used this application to understand the working of Flutter and Firebase together. BrewCrew is an application used to keep track of crew members' coffee preferences. Every new user is alloted with default values for sugar and strength of their coffee, which they can update later. Each user has access to their crew's preferences, this allows one team member to make coffee runs for the team without having to note down/remember preferences every time.

  • CrafTrip: Tinder for travel

    • This is an application my team of 5 and I created as a part of our course project. This application serves as a one stop destination for confused travel enthusiasts who would like to compare two or more travel destinations. The application allows users to swipe through "travel destination cards" and provides the user with essential information about the destination including descriptive tags. Further the application makes travel destination recommendations based on the user's favourites and history!

Python Projects

  • Hangman

    • A simple Python command line application of the game Hangman. The objective of the game is simple: guess the word before you run out of a fixed number of tries The player can see a visual representation of running out of tries.

  • BlogPost

    • A blog page developed using Python's Flask framework. The page allows different users to create an account, post blogs, view other blogs. Users are further allowed to edit or delete their own posts.
